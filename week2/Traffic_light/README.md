# Traffic Light Controller

## Objective

Simulate a traffic light using Arduino UNO with three LEDs.

## Features

- Red ON for 5 seconds
- Yellow ON for 2 seconds
- Green ON for 4 seconds
- Pedestrian button forces RED for 8 seconds
- Current state displayed on Serial Monitor with timestamp

## Components

- Arduino UNO
- 3 LEDs
- 3 × 220Ω Resistors
- Push Button
- 10kΩ Resistor
