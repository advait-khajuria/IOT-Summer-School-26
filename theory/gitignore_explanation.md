# Purpose of .gitignore

The `.gitignore` file tells Git which files and folders should not be tracked by the repository.

## Why it is Important

- Prevents unnecessary files from being committed.
- Excludes temporary files.
- Avoids uploading build files.
- Keeps the repository clean.
- Protects sensitive configuration files.

## Examples

- *.log
- *.tmp
- build/
- .vscode/
- Thumbs.db
